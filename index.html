Multivariate Time-Series Forecast
Libraries Used: Statsmodels, NumPy, Pandas, Scikit-learn, Seaborn.

In this project, I built a Vector Autoregression Model to forecast future values of a multivariate time-series. I performed statistical tests to check the stationarity of the time-series. Using Seaborn, I plotted graphs to show how close my model's forecasts were to the actual values.

TF-IDF Matrix for Text Classification
Libraries Used: NLTK, WordCloud, Pandas, Scikit-learn, NumPy, Seaborn.

I developed a TF-IDF matrix to classify user reviews from Amazon, Yelp and IMDB. A TF-IDF Matrix is a document weighing scheme that assigns weights to words in the text based on their occurence across all documents.

Blog Code

Sentiment Analysis of News Articles in the Strauss-Kahn case
Libraries Used: NLTK, VaDER, Pandas, Scikit-learn, Seaborn, Matplotlib.

Using the NLTK library and VADER sentiment analysis tool, I analyzed multiple news articles covering the sexual abuse allegations against former IMF director, Dominique Strauss-Kahn. I was particularly interested in identifying whether the articles were reporting facts in an objective manner or were solely opinionated and biased in a subjective fashion.
